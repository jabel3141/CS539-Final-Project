Project Models

Parsing Style 1

Model Info

  • Reduced features to 12 per game session
  • Neural Network from MNIST examples
  • 2 Layer with 12 neruons and 4 output values
  • Batch size of 50 and 30 epochs

Results

  • CNN: 17% Accuracy

Downlaod

  • First download and run the MINSTCNN.py file

Parsing Style 2

Model Info

  • Combined label to installation_id and only looked at the last assessment score for training
  • Looked at the last 300 actions a user took
  • Looked at game_time, event_count, event_code (categorical), map_title (categorical), type(categorical)
  • SVM, K nearest neighbors, Random forest, Convolutional Neural Network

Results

  • SVM - 45% accuracy (mostly 3s some 0s)
  • K-Nearest neighbors - 49% accuracy (guessed everything, but mostly 3 or 0)
  • Random forest - 48% accuracy (guessed mostly 3 or 0, and sometimes 1
  • CNN - 51% accuracy (guessed only 3)

Downlaod

  • Download and run the CNN.py file. This has the CNN as well as SVM, KNN, and Random Forest.

Parsing Style 3

Model Info

  • Used only assessment data and combined and sorted event_id by timestamp
  • Limited length of a sequence to 400 steps. Appended 0000 sequences to match those with less than 400 steps
  • Sequential Neural Network with 5 epochs

Results

  • Sequential Neural Network: ~98% while only using assessment data

Downlaod

  • First download the kaggleSubmissionFile.ipynb file. Then run on kaggle competition page. To run the file not for submission, you will just need to change how the program is reading in the in put data for the training, testing and train_labels.

Parsing Style 4

Model Info

  • Any game data played prior to a given assessment is given the same accuracy group as the assessment
  • Aggregated by group number where a group number represents all assessment data with the game data played prior to that given assessment
  • Logistic Regression, K-Nearest Neighbors, Support Vector Machine, Sequential Neural Network

Results

  • Logistic Regression - ~47.5% (Most predictions being either 0 or 3)
  • K-Nearest Neighbors - ~50% (Most predictions being 3)
  • Support Vector Machine - ~50% (Most prediciton being 3)
  • Sequential Neural Network - ~54.5% (Most prediciton being 0 or 3)

Downlaod

  • First download and run the PickleFileCreation.py file
  • Then download and run the Logistic_KNN_SVM.py file for Logistic Regression, KNN, SVM
  • Then download and run the GroupsSequentialModel.py file for Sequential Neural Network
